## 💡 TIPS & TRICKS

### **Tip #1: Utiliser le Logo Partout**
```typescript
import { LogoHeader } from '@/components/LogoHeader';

// Facile! Juste utilisez:
<LogoHeader size="small" />    // 60px
<LogoHeader size="medium" />   // 80px
<LogoHeader size="large" />    // 120px
```

### **Tip #2: Ajouter le Logo à D'autres Screens**
```typescript
// Dans explore.tsx, add-project.tsx, etc:
import { Header } from '@/components/Header';

<Header 
  title="Mon Screen"
  showLogo={true}
  showBackButton={true}
/>
```

### **Tip #3: Redémarrer avec Cache Clear**
```bash
npm start -- --clear
# Si le logo ne s'affiche pas
```

### **Tip #4: Vérifier le Logo Existe**
```bash
# Vérifier que le fichier est présent:
ls assets/logos/logo.png    # Mac/Linux
dir assets\logos\logo.png   # Windows
```

## 🐛 TROUBLESHOOTING

### **Logo ne s'affiche pas?**
✅ Solution:
```bash
npm start -- --clear
# Puis redémarrez l'app complètement
```

### **Logo est blurry?**
✅ Solution:
- Logo PNG doit être 1024x1024px minimum
- Vérifier que c'est un PNG valide

### **App ne compile pas?**
✅ Solution:
```bash
npm install
npm start
# Vérifier la console pour les erreurs
```

### **Couleurs pas exactes?**
✅ Solution:
- Logo PNG doit avoir ses vraies couleurs
- Pas de modification CSS, utiliser les props
